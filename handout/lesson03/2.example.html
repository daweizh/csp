Example 1

如图，在梯形中阴影部分面积是150平方厘米，求梯形面积。

#include <iostream>
#include <cstdio>
#include <cstdlib>
using namespace std;

int main(){
    float s,h,up,down;
    
    up = 15;
    down = 25;
    h = 2* 15 / up;
    s = (up+down)*h/2;
    printf("s=%.2f\n",s);
    
    system("pause");
    
    return 0;
}

Example 3

有一个牧场，牧场上的牧草每天都在匀速生长，这片牧场可供15头牛吃20天，或可供20头牛吃10天，那么，这片牧场每天新生的草量可供几头牛吃1天？

#include <iostream>
#include <cstdio>
using namespace std;

int main(){
    int s1,s2,s3;
    
    s1 = 15 * 20;
    s2 = 20 * 10;
    s3 = (s1-s2)/(20-10);
    cout << "s=" << s3 << endl;
    
    return 0;
}

设：

  • 每头牛每天吃1单位草

则：

  1. 15头牛20天共吃300单位的草
  2. 20头牛10天共吃200单位的草

设

  • 草场原有a单位草
  • 草场每天生长x单位的草

则

  1. a + 20x = 300
  2. a + 10x = 200
  3. x = (300 -200) / (20 - 10)
